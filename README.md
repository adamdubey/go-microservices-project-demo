# go-microservices-project-demo

## Quick Start

_Building & Starting Workloads:_
```sh
# Build & Start container services
make up_build

# Build front-end
make build_front

# Start front-end
make start

# in browser, visit http://localhost
```

_Stopping Workloads:_
```sh
# Stop all container services
make down

# Stop front-end
make stop
```
